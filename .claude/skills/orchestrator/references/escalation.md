# 에스컬레이션 기준

다음 중 하나라도 발생하면 **즉시 작업을 정지**하고 사용자에게 보고한다.
혼자 해결하려 시도하지 않음.

## 자동 차단 조건 (훅이 감지)

- [ ] 자가수정 3회 연속 실패 (`verify-and-block.sh`가 차단)
- [ ] `domain/` 패키지에 Spring/JPA import (ArchUnit이 차단)
- [ ] main/master 브랜치 메인 디렉토리에서 소스 편집 시도 (`enforce-worktree.sh`가 차단)
- [ ] EXEC_PLAN 없이 코드 편집 시도 (`enforce-plan.sh`가 차단)

## 수동 판단 조건 (에이전트가 스스로 감지)

- [ ] EXEC_PLAN과 실제 구현의 diff 규모가 150% 초과 (scope creep)
- [ ] DB 마이그레이션 스크립트가 필요한 변경
- [ ] 공개 API Breaking Change 발견
- [ ] Kafka 이벤트 계약 변경
- [ ] 보안 경계 변경 (인증·인가·PII 처리 방식 변경)
- [ ] 롤백 전략이 불명확한 변경
- [ ] Bounded Context 경계를 넘나드는 예기치 못한 결합 발견
- [ ] 기존 상태기계(3D 생성, 주문 등)에 영향을 주는데 EXEC_PLAN에 명시되지 않았음
